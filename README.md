# compare-regions-jp

日本の地域を複数観点から比較し、差異をMarkdown形式でレポートするCLIツール

## 🚀 クイックスタート

### インストール（開発版）

```bash
# リポジトリをクローン
git clone https://github.com/your-username/compare-regions-jp.git
cd compare-regions-jp

# Poetry環境セットアップ
poetry install

# または requirements.txt使用
pip install -r requirements.txt
```
